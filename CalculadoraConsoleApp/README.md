# CALCULADORA 2026

![]()

## Introdução

Uma calculadora simples mas que permite realizar as quatro operações matemática, além da vizualição do histórico de operações e tabuada.

## Funcionalidades

* **operações básicas:** realiza adição, subtração, multiplicação e divisões.
* **Tratamento de divisão por zero:** A calculadora é capaz de validar erros de divisão por zero.
* **Tabuada:** A calculadora é capaz de gerar a tabuada do número informado.
* **Histórico de operações:** A calculadora é capaz de armazenar um histórico das operações anteriores.

## Como utilizar o programa

1. Clone o repositório ou baixo o código comprimido em .zip.
2. Abra o emulador de terminal e navegue até a pasta raiz.
3. Utilize o comando abaixo para restaurar as dependências do projeto.

     ```
     dotnet restore
     ```

4. Em seguida compile e execute o projeto com o comando: 

    ```
    dotnet ru --project CalculadoraConsoleApp
    ```

## Requistitos

* .NET SDK 10.0
